# Trekko_Hardware

<img src="https://github.com/sbcshop/Trekko_Software/blob/main/images/trekko_banner.jpg">


This GitHub provides hardware design files for the Trekko - Pico GPS Logger.

### Repository Contents:
  - [/Design Data](https://github.com/sbcshop/Trekko_Hardware/tree/main/Design%20Data) - This directory contains Schematic, Top and Bottom View
  - [/Mechanical Data](https://github.com/sbcshop/Trekko_Hardware/tree/main/Mechanical%20Data) - This directory contains Dimension, 3D and STEP files
  - [Trekko Getting Started Guide](https://github.com/sbcshop/Trekko_Software) - This repo contains Trekko Getting Started and Sample Codes.
